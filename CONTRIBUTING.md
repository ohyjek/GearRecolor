# Contributing

Thanks for contributing to `GearRecolor`.

## Local Testing

1. Place this folder at:
   - `World of Warcraft\_retail_\Interface\AddOns\GearRecolor`
2. Launch Retail and enable the addon.
3. Run `/reload` after making edits.
4. Hover upgraded gear and verify tooltip track colors.

## Code Style

- Keep Lua code simple and defensive against API changes.
- Prefer local variables over globals.
- Keep comments concise and focused on non-obvious logic.

## Release Notes

- Update `GearRecolor.toc` (`## Version` and `## Interface` when needed).
- Add a short entry to `CHANGELOG.md`.
- Follow `RELEASE_CHECKLIST.md` before publishing.
